// About

What is VOID?

VOID Game Development Studio was built with ease of use as the primary goal. Using a familiar desktop-based environment, it provides an intuitive interface that feels natural for both beginners and experienced developers.

This project was created primarily as a tool to build my own games. I grew tired of the long wait times of some other engines, having to compile shaders and recompile entire codebases. By having full control over the backend, I can easily debug issues, add features, and customize the engine to fit my specific needs without waiting for third-party updates or workarounds.

Key priorities include small loading times and massive potential through a comprehensive library of built-in modules covering as many different game behaviors as possible — from physics and rendering to AI behaviors, UI systems, and complete game templates.

My hope is that other indie developers will see the potential in this tool and find it useful for their own projects. Whether you're prototyping ideas, learning game development, entering game jams or building your next release — VOID is here to help make that process smoother and more enjoyable.
