# Tower Defense

This is an exported game created with **Void Game Dev Desktop**.

## How to Run

1. Extract all files from this ZIP
2. Open `index.html` in a web browser
3. Click the **Play** button to start the game

## Contents

- `index.html` - The main game file (open this in a browser)
- `files/` - Contains all game assets, scripts, and scenes
  - `scripts/` - Custom game scripts
  - `modules/` - Built-in game modules
  - `scenes/` - Game scenes
  - Assets are stored in their original directory structure

## Note

The game will automatically start when you open `index.html`.  
A loading screen will display progress as assets are loaded.

## System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- WebGL support (for advanced graphics)
